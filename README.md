#Newark, DE Open Data Explorer
An interactive, front-end dashboard that visualizes real-time crime and parks data for Newark, Delaware using open government APIs and modern web technologies.

Features
Live crime data from the Delaware Open Data Portal with monthly trend visualization.​

Crime type distribution chart to see which offenses are most common.​

Parks and recreation overview, including total parks, acres, and facilities.

Responsive, single-page layout with tabbed navigation (Dashboard, Parks, About).​

Export functionality to download crime trend data as CSV or JSON.

Seasonal and neighborhood-level crime insights (designed for future enhancements).

Tech Stack
HTML5 & CSS3 (responsive layout, custom design system)

Vanilla JavaScript (no frameworks)

Chart.js for interactive charts and graphs

Delaware Open Data Portal (Socrata API) as the live data source

Deployed on Vercel (static hosting over global CDN)​
